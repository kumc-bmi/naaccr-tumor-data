NAACCR XML: Tumor.pediatricStage

Description
Code for stage of pediatric tumor in an AJCC stage scheme, a pediatric intergroup study scheme, or a pediatric cooperative group scheme.
Rationale
Staging of pediatric tumors requires very different schemes from those used to stage adult tumors.
Codes
See the ROADS manual for allowable codes for this field.
Note: This data item is no longer supported by CoC (as of January 1, 2003).
